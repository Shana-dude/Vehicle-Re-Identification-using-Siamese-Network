# Vehicle-Re-Identification-using-Siamese-Network
Vehicle Re-Identification using Siamese Network

This project implements a deep learning model from scratch to identify vehicles across multiple camera views.

## Features
- Siamese Network (PyTorch)
- Custom CNN Encoder
- Contrastive Loss
- Feature Embedding + Cosine Similarity

## Results
- Rank-1 Accuracy: 0.70
- mAP: 0.11

## Dataset
VeRi Dataset
